# Discord-Staff-List

> requirement

```
pip install -r requirements.txt
```
# config.json
Add a custome role
```json
  "Roles": {
    "{Role Name}": {
      "RoleID": "{Role ID}"
    }
```
example
```json
  "Roles": {
    "Admin": {
      "RoleID": 1096806224705622297
    },
    "Member": {
      "RoleID": 1025223239653392435
    }
```
